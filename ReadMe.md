### PULL

`팀 탐색기` - `홈` 클릭 - `동기화` - `끌어오기`

![pull](./etc/pull.JPG)

원격 저장소의 내용이 내 컴퓨터의 로컬 저장소로 동기화된다.

<br>

### COMMIT

`팀 탐색기` - `변경 내용`

변경한 내용에 대한 설명(노란색 박스)를 쓰고 - `모두 커밋`

<br>


### PUSH

`팀 탐색기` - `동기화` - 나가는 커밋 - `푸시`

내 로컬 저장소에 변경된 내용이 원격 저장소로 업로드하고,

다른 사람들이 해당 내용을 볼 수 있게 한다.

<br>

<br>


### 오류 뜬 경우(conflict)

아직 오류 안 떠서 설명 불가

오류 뜨면 설명 추가 예정
